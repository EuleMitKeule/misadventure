# Misadventure

You are the hero of the forest.<br>
Explore different areas and find secrets, but most importantly survive!

### Controls

| Action        | Control       |
| ------------- |:-------------:|
| Move          | WASD          |
| Aim           | Mouse         |
| Shoot         | Mouse Left    |
| Change Weapon | Mouse Wheel   |
| Pickup Weapon | E             |
